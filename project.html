What is "Occupy Space"?
This is a student project by Gina Bartzok, which was developed during a course at Hochschule Mainz by Prof. Florian Jennett. This Project focuses on the different languages people use to describe dance. The videos used on this page don't belong to me, but to Nowness.
Why is this page called "Occupy space"?
Well, this project didn't had a name for a long time. But then someone, i don't know who, described one video with these words. When I read it I really liked this description, because to me this matches a lot of motions pretty good.
