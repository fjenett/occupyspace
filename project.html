What is "Occupy Space"?
This is a student project by Gina Bartzok, developed during a course led by Prof. Florian Jennett at Hochschule Mainz. This Project focuses on the different languages people use to describe dance. The videos used on this page don't belong to me, but to Nowness.

Why is this page called "Occupy Space"?
Well, this project didn't have a name for a long time. But then someone, described one video with these specific words. When I read it I really liked the description, because to me this matches a lot of motions well.
